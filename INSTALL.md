# Installation

## From binary package

Go to the [earthlyls release page](https://github.com/glehmann/earthlyls/releases), download the binary package for
your OS, extract it and copy the `earthlyls` (`earthlyls.exe` on Windows) in a directory in your `PATH` â€” for example
`~/.local/bin/`.

## From source package

Just run:

~~~sh
cargo install earthlyls
~~~

## From sources

Clone the earthlyls repository and run:

~~~sh
cargo install --path .
~~~
